## KEY FORMULAS FOR CALCULATORS

### 1. LOAD CALCULATIONS (Manual J Basis)

**Sensible Cooling Load Formula:**
```
BTUh = CFM × ΔT × 1.08

Where:
BTUh = British Thermal Units per hour
CFM = Cubic feet per minute
ΔT = Temperature difference (°F)
1.08 = Constant (density × specific heat × 60 min/hr)
```

**Latent Cooling Load Formula:**
```
BTUh = CFM × ΔW × 0.68

Where:
ΔW = Humidity ratio difference (grains/lb)
0.68 = Latent heat constant
```

**Total Cooling Load:**
```
Total BTUh = Sensible BTUh + Latent BTUh
```

**Equipment Sizing:**
```
Tons of cooling = Total BTUh ÷ 12,000

1 ton = 12,000 BTU/hr
```

---

### 2. AIRFLOW (CFM) CALCULATIONS

**CFM from Air Changes:**
```
CFM = (Room Volume × ACH) ÷ 60

Where:
Room Volume = Length × Width × Height (cubic feet)
ACH = Air changes per hour
```

**CFM from Duct Velocity:**
```
CFM = Velocity × Area

Where:
Velocity = feet per minute (FPM)
Area = duct cross-section (square feet)
```

**CFM per Ton Rule of Thumb:**
```
CFM = Tons × 400

Standard: 400 CFM per ton of cooling
Range: 350-450 CFM/ton depending on conditions
```

**CFM per Square Foot:**
```
Residential cooling: 1 CFM per sq ft
Commercial varies: 0.5-2 CFM per sq ft
```

---

### 3. DUCT SIZING (Manual D Basis)

**Velocity Method:**
```
Duct Area (sq ft) = CFM ÷ Velocity (FPM)

For round duct:
Diameter = √(Area × 4 ÷ π)
```

**Friction Rate Method:**
```
Available Static Pressure = Total External Static Pressure - Component Losses
Friction Rate = Available SP × 100 ÷ Total Equivalent Length

Where:
Friction Rate in inches WC per 100 ft
```

**Equivalent Length:**
```
Total Equivalent Length = Actual Length + Fitting Equivalent Lengths

Common fitting equivalents:
- 90° elbow: 10-15 ft equivalent
- 45° elbow: 5-7 ft equivalent
- Tee (branch): 25-35 ft equivalent
- Register boot: 10-15 ft equivalent
```

**Recommended Duct Velocities:**

| Application | Main Duct (FPM) | Branch (FPM) |
|-------------|-----------------|--------------|
| Residential | 700-900 | 500-700 |
| Commercial | 1200-1500 | 800-1000 |
| Industrial | 1500-2500 | 1000-1500 |

### 4. HEAT GAIN/LOSS CALCULATIONS

**Conduction Heat Transfer:**
```
Q = U × A × ΔT

Where:
Q = Heat transfer (BTU/hr)
U = Overall heat transfer coefficient (BTU/hr·ft²·°F)
A = Area (sq ft)
ΔT = Temperature difference (°F)
```

**R-Value to U-Value:**
```
U = 1 ÷ R

Example: R-19 wall
U = 1 ÷ 19 = 0.053
```

**Infiltration Load:**
```
Sensible: Q = 1.08 × CFM × ΔT
Latent: Q = 0.68 × CFM × ΔW
```

**Solar Heat Gain:**
```
Q = A × SHGC × Solar Intensity × Shading Factor

Where:
SHGC = Solar Heat Gain Coefficient
```

---

### 5. PSYCHROMETRIC CALCULATIONS

**Relative Humidity:**
```
RH = (Actual Vapor Pressure ÷ Saturation Vapor Pressure) × 100%
```

**Dew Point Approximation:**
```
Td ≈ T - ((100 - RH) ÷ 5)

Where:
Td = Dew point (°F)
T = Dry bulb temperature (°F)
RH = Relative humidity (%)
```

**Wet Bulb Depression:**
```
WB Depression = Dry Bulb - Wet Bulb

Used to estimate humidity
```

**Enthalpy (Total Heat):**
```
h = 0.24T + W(1061 + 0.444T)

Where:
h = Enthalpy (BTU/lb dry air)
T = Temperature (°F)
W = Humidity ratio (lb water/lb dry air)
```

---

### 6. REFRIGERATION CALCULATIONS

**Refrigeration Effect:**
```
RE = h₁ - h₄

Where:
h₁ = Enthalpy leaving evaporator
h₄ = Enthalpy entering evaporator
```

**Coefficient of Performance (COP):**
```
COP = Cooling Effect ÷ Work Input

Or: COP = (h₁ - h₄) ÷ (h₂ - h₁)
```

**Energy Efficiency Ratio (EER):**
```
EER = Cooling capacity (BTU/hr) ÷ Power input (Watts)

Example: 12,000 BTU ÷ 1,000W = 12 EER
```

**Seasonal Energy Efficiency Ratio (SEER):**
```
SEER = Total cooling (BTU) ÷ Total energy (Wh) over season

SEER2 (2023+) = Measured with higher static pressure
```

**SEER to EER Conversion (Approximate):**
```
EER ≈ SEER × 0.875 (for older units)
EER ≈ SEER × 0.9 (for newer high-efficiency)
```

**Minimum Efficiency Standards (2023+):**

| Region | AC SEER2 Min | Heat Pump SEER2 Min |
|--------|--------------|---------------------|
| North | 14.3 | 14.3 |
| South | 15.2 | 15.2 |
| Southwest | 15.2 | 15.2 |

### 7. HYDRONIC HEATING CALCULATIONS

**Heat Transfer Rate:**
```
Q = 500 × GPM × ΔT

Where:
Q = Heat transfer (BTU/hr)
500 = Constant for water (8.33 lb/gal × 60 min/hr)
GPM = Gallons per minute
ΔT = Temperature difference (°F)
```

**Pump Head Calculation:**
```
Head (ft) = Friction Loss + Static Head + Fitting Losses

Friction Loss = f × (L/D) × (V²/2g)
```

**Flow Rate for Required Heat:**
```
GPM = BTU/hr ÷ (500 × ΔT)

Example: 50,000 BTU with 20°F ΔT
GPM = 50,000 ÷ (500 × 20) = 5 GPM
```

---

### 8. VENTILATION CALCULATIONS (ASHRAE 62.2)

**Residential Ventilation Rate:**
```
Qtot = 0.03 × Afloor + 7.5 × (Nbr + 1)

Where:
Qtot = Total ventilation (CFM)
Afloor = Floor area (sq ft)
Nbr = Number of bedrooms
```

**Minimum Continuous Exhaust:**
```
Kitchen: 100 CFM intermittent or 25 CFM continuous
Bathroom: 50 CFM intermittent or 20 CFM continuous
```

**Air Changes Per Hour by Space Type:**

| Space Type | ACH Range |
|------------|-----------|
| Residential (general) | 0.35 min |
| Office | 4-6 |
| Conference room | 6-8 |
| Restaurant dining | 8-10 |
| Kitchen (commercial) | 15-20 |
| Bar/lounge | 15-20 |
| Hospital OR | 15-25 |
| Clean room | 20-600 |

### 9. GAS FURNACE SIZING

**Input vs Output:**
```
Output BTU = Input BTU × AFUE

Where:
AFUE = Annual Fuel Utilization Efficiency (%)

Example: 100,000 BTU input × 0.95 AFUE = 95,000 BTU output
```

**Temperature Rise:**
```
ΔT = Output BTU ÷ (1.08 × CFM)

Must be within manufacturer's range (typically 30-60°F)
```

**CFM for Furnace:**
```
CFM = Output BTU ÷ (1.08 × ΔT)

Example: 80,000 BTU ÷ (1.08 × 50°F) = 1,481 CFM
```

**Gas Pipe Sizing:** (See Plumbing INTEL)

### 10. HEAT PUMP CALCULATIONS

**Heating Capacity vs Temperature:**
```
Heat pumps lose capacity as outdoor temp drops
Typical: 2-3% capacity loss per °F below 47°F
```

**Balance Point:**
```
Balance Point = Temperature where heat pump capacity equals building load

Below balance point: supplemental heat required
```

**COP vs Temperature:**

| Outdoor Temp | Typical COP |
|--------------|-------------|
| 47°F | 3.0-4.0 |
| 35°F | 2.5-3.0 |
| 17°F | 2.0-2.5 |
| 0°F | 1.5-2.0 |

**Defrost Cycle:**
```
Reduces effective capacity by 5-15%
More frequent below 35°F outdoor
```

## EPA 608 CERTIFICATION

### Certification Types

| Type | Covers | Equipment Examples |
|------|--------|-------------------|
| **Type I** | Small appliances (≤5 lbs refrigerant) | Window AC, refrigerators, PTAC, vending |
| **Type II** | High-pressure refrigerants | Residential AC, heat pumps, supermarket |
| **Type III** | Low-pressure refrigerants | Centrifugal chillers |
| **Universal** | All types (I + II + III) | Full HVAC service |

### Exam Requirements

| Section | Questions | Passing Score |
|---------|-----------|---------------|
| Core | 25 | 70% (proctored) / 84% (open) |
| Type I | 25 | 70% / 84% |
| Type II | 25 | 70% / 84% |
| Type III | 25 | 70% / 84% |

**Note:** Universal requires proctored Core exam

### Key EPA 608 Topics

1. **Ozone Depletion** - CFC/HCFC phase-out
2. **Refrigerant Recovery** - Required before service
3. **Leak Repair** - Mandatory for systems >50 lbs
4. **Record Keeping** - Required for commercial
5. **Venting Prohibition** - Illegal to vent refrigerants
6. **Safe Disposal** - Equipment disposal requirements

| A3 | Higher flammability | Low | R-290 (propane) |
