## KEY FORMULAS FOR CALCULATORS

### 1. DRAINAGE FIXTURE UNITS (DFU)

**Definition:**
```
1 DFU = Flow equivalent of 1 cubic foot of water drained through 1-1/4" pipe in 1 minute
1 DFU ≈ 7.48 GPM ≈ 0.47 L/s
```

**Common DFU Values (IPC Table 709.1):**

| Fixture | DFU | Min Trap Size |
|---------|-----|---------------|
| Bathtub | 2 | 1-1/2" |
| Bidet | 1 | 1-1/4" |
| Clothes washer (residential) | 2 | 2" |
| Clothes washer (commercial) | 3 | 2" |
| Dishwasher | 2 | 1-1/2" |
| Floor drain | 2 | 2" |
| Kitchen sink (residential) | 2 | 1-1/2" |
| Kitchen sink (commercial) | 3 | 2" |
| Lavatory (private) | 1 | 1-1/4" |
| Lavatory (public) | 2 | 1-1/4" |
| Shower | 2 | 2" |
| Sink (service/mop) | 3 | 3" |
| Urinal (flushometer) | 4 | 2" |
| Urinal (flush tank) | 2 | 1-1/2" |
| Water closet (private, tank) | 3 | 3" |
| Water closet (public, tank) | 4 | 3" |
| Water closet (flushometer) | 6 | 3" |

**UPC vs IPC DFU Differences:**

| Fixture | IPC DFU | UPC DFU |
|---------|---------|---------|
| Water closet (public, flushometer) | 6 | 4 |
| Lavatory (public) | 2 | 1 |
| Urinal (flushometer) | 4 | 4 |

### 4. WATER PIPE SIZING

**Minimum Pipe Sizes (IPC):**

| WSFU | Min Pipe Size | Notes |
|------|---------------|-------|
| 0-7 | 1/2" | Individual fixture supply |
| 8-14 | 3/4" | Branch line |
| 15-22 | 1" | Branch line |
| 23-40 | 1-1/4" | Main or branch |
| 41-60 | 1-1/2" | Main |
| 61-100 | 2" | Main |
| 101-200 | 2-1/2" | Main |
| 201-400 | 3" | Main |

**Velocity Limits:**
```
Maximum recommended velocity: 8 fps (feet per second)
Higher velocities cause:
- Water hammer
- Pipe erosion
- Noise
```

**Pressure Loss Formula:**
```
For copper pipe (Hazen-Williams C=130):
ΔP = (4.52 × Q^1.85) / (C^1.85 × d^4.87)

Where:
ΔP = Pressure drop (psi per 100 ft)
Q = Flow rate (GPM)
C = Hazen-Williams coefficient
d = Inside diameter (inches)
```

### 5. VENT PIPE SIZING

**Vent Sizing Formula:**
```
Vent size based on:
1. DFU load on drain
2. Length of vent
3. Size of drain served
```

**Individual Vent Sizing (IPC Table 906.1):**

| Drain Size | Max Vent Length | Vent Size |
|------------|-----------------|-----------|
| 1-1/4" | 45 ft | 1-1/4" |
| 1-1/2" | 60 ft | 1-1/4" |
| 2" | 120 ft | 1-1/2" |
| 3" | 212 ft | 2" |
| 4" | 300 ft | 2" |

**Minimum Vent Sizes:**
```
- Individual vent: 1-1/4" minimum (1-1/2" recommended)
- Branch vent: 1-1/2" minimum
- Vent stack: Half size of drain stack (min 1-1/4")
- Stack vent: Same size as drain stack
```

### 6. WATER HEATER SIZING

**Storage Tank Sizing Formula:**
```
Tank Size (gallons) = Peak Hour Demand × First Hour Rating Factor

Peak Hour Demand varies by household:
- 1-2 people: 30-40 gallons
- 3-4 people: 40-50 gallons
- 5+ people: 50-80 gallons
```

**First Hour Rating (FHR):**
```
FHR = Tank capacity + Recovery rate per hour

Recovery Rate (gas):
BTU/hr ÷ 8.33 lb/gal ÷ Temperature Rise = GPH

Example (40,000 BTU, 70°F rise):
40,000 ÷ 8.33 ÷ 70 = 68.6 GPH
```

**Tankless Sizing Formula:**
```
Flow Rate (GPM) needed = Sum of all fixtures that may run simultaneously

Temperature Rise = Desired Temp - Incoming Temp

BTU needed = GPM × 8.33 × Temp Rise × 60 minutes

Example (3 GPM, 70°F rise):
3 × 8.33 × 70 × 60 = 104,916 BTU/hr ≈ 105,000 BTU
```

### 7. GAS PIPE SIZING

**Sizing Factors:**
```
1. Total BTU load (connected appliances)
2. Length of piping from meter
3. Pressure drop allowed (usually 0.5" WC)
4. Specific gravity of gas (natural gas = 0.60)
```

**Common BTU Loads:**

| Appliance | BTU/hr |
|-----------|--------|
| Gas range | 65,000 |
| Tankless water heater | 150,000-200,000 |
| Storage water heater | 30,000-75,000 |
| Furnace | 50,000-150,000 |
| Gas dryer | 22,000-35,000 |
| Fireplace (gas logs) | 25,000-50,000 |
| Pool heater | 150,000-400,000 |

### 8. SEPTIC TANK SIZING

**Minimum Septic Tank Sizes:**

| Bedrooms | Min Tank Size (gallons) |
|----------|-------------------------|
| 1-2 | 750 |
| 3 | 1,000 |
| 4 | 1,200 |
| 5-6 | 1,500 |
| 7-8 | 2,000 |

**Drain Field Sizing Formula:**
```
Square Feet = Daily Flow (GPD) ÷ Application Rate

Daily Flow (residential) = 150 GPD/bedroom

Application Rate varies by soil:
- Gravel/sand: 1.2 GPD/sq ft
- Sandy loam: 0.8 GPD/sq ft
- Clay loam: 0.4 GPD/sq ft
- Clay: 0.2 GPD/sq ft
```

### 9. RAINWATER/GUTTER SIZING

**Roof Drainage Area Formula:**
```
Drainage Area (sq ft) = Length × Width × Pitch Factor

Pitch Factors:
- Flat to 3:12 = 1.0
- 4:12 to 5:12 = 1.05
- 6:12 to 8:12 = 1.10
- 9:12 to 11:12 = 1.20
- 12:12 = 1.40
```

**Gutter Sizing (IPC Table 1106.2):**

| Gutter Size | Max Roof Area (sq ft) | Slope 1/16"/ft |
|-------------|----------------------|----------------|
| 3" | 340 | |
| 4" | 720 | |
| 5" | 1,250 | |
| 6" | 1,920 | |
| 7" | 2,760 | |
| 8" | 3,760 | |

**Downspout Sizing:**
```
Rule: 1 sq inch of downspout per 100 sq ft roof area

Common sizes:
- 2" × 3" (6 sq in) = 600 sq ft roof
- 3" × 4" (12 sq in) = 1,200 sq ft roof
- 4" round (12.6 sq in) = 1,260 sq ft roof
```

### 10. WATER PRESSURE CALCULATIONS

**Minimum Pressures Required:**

| Fixture Type | Min Pressure (psi) |
|--------------|--------------------|
| Lavatory | 8 |
| Shower | 8 |
| Bathtub | 8 |
| Toilet (tank) | 8 |
| Toilet (flushometer) | 15-25 |
| Dishwasher | 8 |
| Clothes washer | 8 |
| Hose bibb | 8 |

**Pressure Loss per Vertical Foot:**
```
Elevation loss = 0.433 psi per foot of height

Example: 20 ft elevation change
20 × 0.433 = 8.66 psi loss
```

**Available Pressure Formula:**
```
Available Pressure = Supply Pressure - (Static Loss + Friction Loss + Meter Loss)

Where:
- Static Loss = 0.433 psi × height (ft)
- Friction Loss = from pipe sizing tables
- Meter Loss = 5-15 psi typical
```

## IPC vs UPC KEY DIFFERENCES

| Topic | IPC | UPC |
|-------|-----|-----|
| **DFU for WC (public flushometer)** | 6 DFU | 4 DFU |
| **Trap arm length** | Varies by table | 24× pipe diameter |
| **Wet venting** | More restrictive | More flexible |
| **Circuit venting** | Allowed | Not in base code |
| **Greywater systems** | Appendix C | Green Plumbing Appendix |
| **Code development** | ICC open process | IAPMO consensus |
| **Interpretive authority** | Prescriptive | More official discretion |
