# LOW VOLTAGE TECHNICIAN
## ZAFTO Intelligence File
### Last Updated: February 1, 2026

---

## OVERVIEW

**What they do:** Install, maintain, and repair low voltage systems - fire alarm, security/access control, structured cabling (data/voice), audio/video, nurse call, and building automation systems.

**Market size:** ~150,000+ low voltage technicians in US, rapid growth with smart buildings and IoT.

**Why it matters for ZAFTO:** Licensed in many states, NICET certification path, overlaps with electrical but distinct trade.

**Calculator target:** 60-75 calculators

## KEY FORMULAS & CALCULATIONS

### Fire Alarm Circuits (NEC 760)

**Voltage Drop (SLC/IDC)**
```
Vd = (2 × K × I × D) / cmil

Where:
- Vd = Voltage drop (V)
- K = 12.9 (copper)
- I = Current (A)
- D = Distance (ft)
- cmil = Circular mils

Max 10% for fire alarm
```

**Wire Resistance**
```
R = ρ × L / A

Common resistances (ohms/1000 ft):
#22 = 16.5
#18 = 6.5
#16 = 4.1
#14 = 2.6
```

**Power Limited Circuits**
```
Class 2: Max 100 VA, 30V max (varies)
Class 3: Max 100 VA, higher voltage allowed
Fire Alarm Power Limited: Per UL listing
```

### Structured Cabling

**Cable Length Limits**
```
Horizontal: 90m (295 ft) permanent link
Patch cords: 10m total (both ends)
Total channel: 100m (328 ft)
```

**Signal Loss Budget**
| Component | Cat5e/Cat6 Loss |
|-----------|-----------------|
| Cable | 2.0 dB/100m typical |
| Connector | 0.2-0.5 dB each |
| Splice | 0.3 dB each |

**Fiber Optic Loss**
```
Multimode (850nm): 3.5 dB/km
Multimode (1300nm): 1.5 dB/km
Singlemode (1310nm): 0.5 dB/km
Singlemode (1550nm): 0.4 dB/km
Connector: 0.5-0.75 dB each
Splice: 0.1-0.3 dB each
```

**Link Budget**
```
Loss Budget = Tx Power - Rx Sensitivity - Safety Margin

Total Loss = (Length × dB/km) + (Connectors × loss) + (Splices × loss)
```

### Audio Systems

**Speaker Wire Sizing**
```
Wire gauge based on:
- Total wattage
- Distance
- Impedance

70V/100V systems: use transformer taps
```

**dB Calculations**
```
dB = 10 × log₁₀(P₂/P₁) for power
dB = 20 × log₁₀(V₂/V₁) for voltage

3 dB = double power
6 dB = double voltage
10 dB = perceived 2× louder
```

### Security Systems

**Camera Storage**
```
Storage (GB) = Bitrate (Mbps) × Time (hrs) × Cameras × 0.45

Or: Bitrate × 3600 seconds × cameras / 8000 = GB/hr
```

**PoE Power Budget**
```
PoE Standards:
- 802.3af: 15.4W per port (12.95W available)
- 802.3at (PoE+): 30W per port (25.5W available)
- 802.3bt (PoE++): 60W or 100W per port
```

## NOTES

- Fire alarm licensing is almost universal
- Security/access control often separate license from fire alarm
- Structured cabling may not require license but BICSI cert expected
- Significant overlap with electrical but different NEC articles
- IT/networking knowledge increasingly important
- PoE and IP-based systems are dominant now
- Integration skills valuable (fire + security + AV + BAS)
- Many techs specialize in one area (fire, data, security, AV)
