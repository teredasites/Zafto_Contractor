# ZAFTO — Trades Application

Cross-platform mobile and web application for the ZAFTO contractor platform.

## Structure

- `lib/` — Flutter/Dart mobile application (iOS, Android, Web, Windows)
- `web-portal/` — Contractor CRM (Next.js) — zafto.cloud
- `team-portal/` — Employee field portal (Next.js) — team.zafto.cloud
- `client-portal/` — Homeowner portal (Next.js) — client.zafto.cloud
- `ops-portal/` — Operations dashboard (Next.js) — ops.zafto.cloud
- `supabase/` — Database migrations and Edge Functions

See the root [README](../../README.md) for full architecture details.
